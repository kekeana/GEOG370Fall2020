In this project I learned how to display images in different projections

Projection Process using QGIS

Starting from the Globe Builder plug in, I created a world map layer. Then I added the Indicatrix mapper as layers of graticule and caps, placing those layers above the map. Clicking the Project Coordinate Reference System (CRS) from the bottom right corner, I toggled between the different CRS available.
